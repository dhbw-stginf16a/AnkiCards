# AnkiCards

Using http://reh.math.uni-duesseldorf.de/~zibrowius/LatexNoteImporter/

Basic example;

```
\begin{document}
\begin{note}
   \begin{field}
       Is it true that \(1 + 1 = 0\)?
   \end{field}
   \begin{field}
       Yes and no. The equation
       \[
            1+1 = 0
       \]
       holds only in characteristic two.
   \end{field}
\end{note}
\end{document}
```
